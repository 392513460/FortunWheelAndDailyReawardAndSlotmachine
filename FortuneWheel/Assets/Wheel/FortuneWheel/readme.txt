2D version of "Fortune Wheel" for your games. 
Free/paid spins with a timer. Useful for monetisation and daily bonuses.
Prizes and win items may be provided by a server-side. 
This project may be used as a base for new games.

Features:
+ Ready project with screens and popups in 2D Canvas
+ JSON settings of wheel with Local/Remote control
+ Mobile ready
+ Easy setup of costs, rewards and chances
+ A target sector may be random, based on chance or set by logic in your scripts

In a package:
+ Fortune Wheel controller
+ Client/Server communication controller
+ Localisation controller
+ Popups controller
+ Sound effects controller
+ Resources manager

Dependencies:
+ DOTween
+ JSONObject

Unity 5.5.0+ compatible
